Bot Board II

  • 5.5 - 9V logic circuit
  • 4.8 - 7.2V servos

This is the brain of the robot. Right now it has a wall outlet, but when this thing is ready to move it'll need power. It also needs a DB9 to IDC10 ribbon cable to connect to the servocontroller. This is a serial connection that sends strings of instructions that select the: servo, position, and speed. Linux treats everything as a file, so sending these instructions should be as simple as reading and writing to a file.

Beagleboard Rev C2

  • ArchLinux ARM
  • JTIG
  • EHCI (2.0 only) USB
  • USB OTG

This is the servocontroller. We need to figure out how we want to move the arm or at least what motion, position, and speed combinations are useful so we can feed that information into the program when we reach that stage. If you're interested, go to lynxmotion.com, Products, Software to download the ArmControl, Rover Controller, and Servo Sequencer programs. Before we can use it, however, it needs power. It's logic circuit is powered by a 9V battery, but the servos need 4.8 - 7.2V.

General command format: #<ch>P<pw>S<spd>...T<time>\r

  • ch = servo channel
  • pw = pulse width = position
  • spd = max speed (if slower, this supersedes time)
  • This pattern can be repeated for multiple servos in one action
  • time = max time for all servos to complete the action
  • all commands end with carriage return

SSC-32 Servo Controller

  • 5.5 - 9V logic circuit
  • 4.8 - 7.2V servos
  • Electrically Eraseable Programmable Read-Only Memory
    • The EEPROM is the small removeable chip behind the DB9 serial port.
    • It can store servo sequences.

This is where the motor controller comes in. It has a 5V 1A output which should power the servos, but - it also needs power. These are some of the recommended power sources. As you can see it's not too picky. It has a lithium battery mode to prevent cell damage due to over-discharge. It's strongly recommended a battery connector is used, both for easy removal when charging and to avoid attaching it backwards. There is no reverse polarity protection.

Sabertooth 2x12 Motor Controller

  • 6 - 24V input
  • 5V 1A output
  • Recommended Power Sources:
    • 5 - 18 cell NiMH or NiCd
    • 2s - 6s lithium ion or lithium polymer
    • 6 - 24V lead acid
    • 6 - 24V power supply (when in parallel with a suitable battery)
